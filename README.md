# Enterprise Multi-Domain Data Platform on AWS

This project demonstrates an enterprise-grade data engineering platform using AWS Glue and
Lakehouse architecture (Bronze, Silver, Gold).

## Architecture
- Bronze: Raw data ingestion with metadata
- Silver: Cleansing, validation, deduplication, business rules
- Gold: Analytics-ready aggregated datasets

## Tech Stack
AWS Glue, PySpark, Amazon S3, Step Functions, Lambda, Athena

## Key Features
- Multi-domain processing
- Schema evolution support
- Enterprise-grade transformations
- Scalable and cost-optimized design
